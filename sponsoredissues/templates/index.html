What is sponsoredissues.org?
sponsoredissues.org lets GitHub Sponsors assign their donations to specific GitHub issues, in order to vote for which features and fixes get prioritized.
The ability to assign donations to specific GitHub issues has two benefits:
  1. It gives FOSS users an additional incentive to donate, because it gives them voting power.
  2. It helps FOSS maintainers understand which issues are most important to their users.
For more detailed info about how the site works, please see the FAQ.
For help getting started as a FOSS maintainer, see: How do I get started as a maintainer?
For help getting started as a FOSS user/donor, see: How do I vote for issues on sponsoredissues.org?
This website is free to use, and the source code is published on GitHub with an Aferro GPL license. If want to support the site, or if you want to vote for future features/fixes, consider donating to me on GitHub Sponsors.
